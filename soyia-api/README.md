## Passos no projeto

1. Instalar virtualenv

`pip install virtualenv`

2. Criar ambiente do virtualenv

`virtualenv myenv`

3. Ativar virtualenv

`.\myenv\Scripts\activate`

4. Instalar dependencias

`pip install -r requirements.txt`

6. Rodar app

`python server.py`
